Libspire_serial

Library for communication with serial ports written in Vala (Windows - Linux)

Download .zip Download .tar.gz View on GitHub

Welcome to libspire_serial.

This library allows communication with devices right through serial / USB ports such as modems, sensors, bar code readers, etc.

The library is very simple and useful, written in Vala programming language that allowed me to compile without problem for both Linux and Windows.
