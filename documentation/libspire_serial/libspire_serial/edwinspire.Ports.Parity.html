Description:

public enum Parity

Specifies the parity bit for the serial communication


Namespace: edwinspire.Ports
Package: libspire_serial

Content:

Enum values:

  • NONE - Without testing calculation parity bit
  • ODD - Sets the parity bit so that the count of bits set is an odd number
  • EVEN - Sets the parity bit so that the count of bits set is an even number
  • MARK - Sets the parity bit set to 1
  • SPACE - Sets the parity bit set to 0
