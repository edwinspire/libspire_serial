public enum HandShaking

Specifies the control protocol used to establish communication with the serial port


Namespace: edwinspire.Ports
Package: libspire_serial

Content:

Enum values:

  • NONE - No control is not used for the protocol binding
  • RTS_CTS - Controls hardware, Ready to Send (RTS) and Clear to Send (CTS)
  • XOnXOff - Protocol software control XON / XOFF
  • DTR_DSR - Use both RTS CTS and XON / XOFF
